CSS Transitions

Gaétan RENAUDEAU

Les CSS Transitions sont partie intégrante du CSS3. Ils permettent de faire évoluer les valeurs de propriétés CSS dans une durée définie. C'est un moyen simple pour enrichir l'Interface Utilisateur d'une application web : quelques lignes de plus dans les feuilles de style. Il n'est souvent plus nécessaire d'utiliser des technologies plus lourdes et non web comme Flash ou Silverlight. Les possibilités sont suffisantes pour la plupart des besoins couplés avec les autres nouveautés du CSS3 et parfois avec un peu de Javascript. Les CSS Transitions sont désormais dans les standards et fonctionnent dans tous les navigateurs modernes, IE9 compris.

Pourquoi WebUI

Guillaume BORT

Si on peut démontrer qu'aujourd'hui HTML5 et les technologies 100% Web sont capables de faire aussi bien que flash pour les applications à contenu riche, il est légitime de se demander quels sont ses avantages réel? Dans cette présentation nous nous intéresseront aux différences d'architectures entre le Web et les technologies basées sur Flash, pour montrer comment créer une application riche en utilisant les standard Web est non seulement possible mais également plus simple et avantageux.
